# ProjetLibrairieEnLigne
Est un projet codé en HTML5/CSS3/PHP7. C'est une librairie en ligne permettant d'enregistrer un compte, voir ses informations saisies, pouvoir les modifier. Une fois le compte
créé, on peut avoir accès aux produits (CD, livres, films) et les réserver. Il n'y a cependant, pas de paiement en ligne. Il y a également, un panel administrateur permettant
de voir les utilisateurs présent dans la base de données SQL, les modifier ou les supprimer. Pour réaliser ce projet, notre tuteur nous a laissé un délai de 4 semaines.
